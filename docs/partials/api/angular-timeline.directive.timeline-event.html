timeline-event
directive in module angular-timeline

Description

Represents an event occuring at a point in time, displayed on the left or the right of the timeline line.

Usage

as element:
<timeline-event
       [side="{string}"]
       [distribute="{string}"]>
</timeline-event>
as attribute
<ANY timeline-event
     [side="{string}"]
     [distribute="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
side
(optional)
string

Define if the side of each elements (ie side="'right'")

distribute
(optional)
string

Define the alternate distribution (ie distribution="'left'")

You typically embed a timeline-badge and timeline-panel element within a timeline-event.
